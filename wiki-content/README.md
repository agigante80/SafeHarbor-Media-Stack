# Wiki Content Structure

This directory contains all the content for the GitHub Wiki, organized by topic.

## Wiki Pages Structure

1. **Home.md** - Wiki homepage and overview
2. **Installation-Guide.md** - Detailed setup instructions  
3. **Configuration.md** - Environment variables and advanced settings
4. **Telegram-Bot.md** - Complete bot setup and functionality
5. **Troubleshooting.md** - Real-world scenarios and solutions
6. **VPN-Monitoring.md** - DNS leak detection and status tracking
7. **Architecture.md** - Technical deep-dive and networking
8. **Synology-Guide.md** - NAS-specific instructions and tips

## How to Use

1. Copy each .md file content to the corresponding GitHub Wiki page
2. Upload any images to the wiki's assets
3. Update internal links to match your repository structure
4. Enable wiki in your repository settings
